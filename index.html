TAUC

Welcome to TAUC

TAUC is a toolset to generate test cases addressing software timeliness. The toolset has been developed by Chunhui Wang, at the SVV lab of the University of Luxembourg.

Requirements

  • Mac OS X / Linux
  • Java 1.7+
